# 错误码<a name="ZH-CN_TOPIC_0022473689"></a>

## 功能说明<a name="section6144247617754"></a>

所有的扩展接口发生400错误时，会有自定义错误信息返回，该节为您介绍每个错误码的含义。

## 返回体格式<a name="section2995295617754"></a>

```
STATUS CODE 400
```

```
{
    "error": {
        "message": "The imagetype is invalid.",
        "code": "IMG.0024"
    }
}
```
