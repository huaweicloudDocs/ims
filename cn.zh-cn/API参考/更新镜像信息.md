# 更新镜像信息<a name="ZH-CN_TOPIC_0020091567"></a>

## 功能介绍<a name="section16095919"></a>

更新镜像信息接口，主要用于镜像属性的修改。

>![](public_sys-resources/icon-note.gif) **说明：**   
>当前仅支持可用（active）状态的镜像更新相关信息。
