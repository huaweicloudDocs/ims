# 镜像属性<a name="ZH-CN_TOPIC_0020091562"></a>

## 概念<a name="section63142895155254"></a>

镜像是一个包含了软件及必要配置的云服务器模版，至少包含操作系统，还可以包含应用软件（例如数据库软件）和私有软件。镜像是镜像管理服务（IMS）管理的核心对象。
