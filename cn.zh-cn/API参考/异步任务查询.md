# 异步任务查询<a name="ZH-CN_TOPIC_0022473688"></a>

## 功能介绍<a name="section6534076917543"></a>

该接口为扩展接口，主要用于查询异步接口执行情况，比如查询导出镜像任务的执行状态。

## URI<a name="section5323664117543"></a>

GET /v1/\{project\_id\}/jobs/\{job\_id\}
